# Memento Web Service  

1. 실행 방법
터미널 (1) 
cd "Memento-Web-Service-main\backend"
npm run dev 

터미널 (2)
cd "Memento-Web-Service-main\frontend"
npm run dev 

주소 입력
http://localhost:5173/

2. 기본 정보
관리자 계정 ID : admin 
관리자 계정 비밀번호 : 1111

- 백엔드
localhost:4000

- DB 확인하기 (터미널에 입력)
cd "Memento-Web-Service-main\backend"
sqlite3 memento.db
sqlite3 database.db (수정해도 반영 안된다면)

3. 수정할 / 개선할 / 확인할 부분
- 목표한 모든 기능이 반영되었는가
- 모든 기능이 정상적으로 작동하는가
- 프론트가 자연스럽게 이어지는가
- 분위기에 맞게 프론트 수정 등 자유롭게 판단
