# Memento Web Service  
AI 라이프 솔루션 챌린지 2025 – “메멘토” 웹 서비스 개발 리포지토리

## 초기 세팅 가이드

- VS Code 확장 기능 설치:
  - **Thunder Client** → API 테스트 (로그인 테스트 등)
  - **SQLite Viewer** → SQLite DB 구조 및 데이터 GUI로 확인
- **Node.js 설치 필수**

---

## 실행 가이드

### 1. `node_modules` 폴더 생성
- npm install → `node_modules` 폴더가 자동 생성됩니다.
- `backend` 폴더와 `frontend` 폴더 각각에서 총 두번 실행합니다.


### 2. 개발 서버 실행
- npm run dev → 서버가 실행됩니다.
- `backend` 폴더와 `frontend` 폴더 각각에서 총 두번 실행합니다. (터미널 창 2개)
  
### 3. 개발 서버 실행
- http://localhost:5173/ 주소로 이동합니다.

### 4. 데이터베이스 확인
- **SQLite Viewer** 확장 프로그램으로 `memento.db` 파일을 GUI 방식으로 열어 테이블과 데이터를 확인할 수 있습니다.
