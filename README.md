# Memento Web Service  
AI 라이프 솔루션 챌린지 2025 – “메멘토” 웹 서비스 개발 리포지토리

---

## 개발 방식 (필독)

- 각 팀은 **자기 팀 브랜치에서만 개발**합니다.
- node_modules 폴더는 올리지 않습니다. 각자 환경에서 각자 운영체제에 맞게 설치합니다.
- main은 절대 수정 금지.
- 디자인은 모든 기능이 일정 수준 완성된 후에 통일해서 적용합니다.
